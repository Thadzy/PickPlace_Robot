จึงถือว่าแบบจำลองมีความแม่นยำเพียงพอสำหรับการออกแบบ
Controller ในย่านการใช้งานจริง

\subsubsection{Unit Test: การอ่านค่า Encoder และ PWM (Deliverable 2)}
\label{subsubsec:unit_test_encoder_pwm}
% ------------------------------------------------------------

การทดสอบนี้มีวัตถุประสงค์เพื่อยืนยันว่า STM32G474RE
สามารถอ่านค่าตำแหน่งเชิงมุมจาก AMT-103 Encoder
ผ่าน Timer ในโหมด Quadrature Encoder Interface (QEI)
ได้อย่างถูกต้อง และสามารถส่งสัญญาณ PWM ไปยัง
Motor Driver Cytron MD10C เพื่อควบคุมความเร็วและ
ทิศทางของมอเตอร์ได้

\paragraph{การตั้งค่า Encoder}

Encoder AMT-103 ตั้งค่าที่ [PPR] Pulses Per Revolution
บน Output shaft ของ Azimuth Joint โดยตรง
เมื่อนับแบบ 4x Quadrature ได้ [COUNTS\_PER\_REV] counts/rev
การแปลงจาก Encoder count เป็นมุมคำนวณได้ดังนี้

\begin{equation}
  \theta_{arm}\ (\text{deg})
  = \frac{\text{count}}{[\text{COUNTS\_PER\_REV}]} \times 360°
\end{equation}

\paragraph{ผลการทดสอบความถูกต้องของ Encoder}

ทดสอบโดยหมุนแขนกลด้วยมือไปยังตำแหน่งอ้างอิงที่กำหนด
แล้วอ่านค่า count จาก Serial Monitor เปรียบเทียบกับมุมจริง
ที่วัดด้วย Protractor
